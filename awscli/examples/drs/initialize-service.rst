**To initialize AWS Elastic Disaster Recovery**

The following ``initialize-service`` example initializes the AWS Elastic Disaster Recovery service in the current Region. ::

    aws drs initialize-service

This command produces no output.

For more information, see `Initializing AWS Elastic Disaster Recovery <https://docs.aws.amazon.com/drs/latest/userguide/getting-started-initializing.html>`__ in the *AWS Elastic Disaster Recovery User Guide*.
