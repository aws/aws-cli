**To delete a recovery instance**

The following ``delete-recovery-instance`` example deletes the specified recovery instance. ::

    aws drs delete-recovery-instance \
        --recovery-instance-id s-1234567890abcdef0

This command produces no output.

For more information, see `Managing recovery instances <https://docs.aws.amazon.com/drs/latest/userguide/recovery-instances.html>`__ in the *AWS Elastic Disaster Recovery User Guide*.
